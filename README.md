# API_second
Task Manager
מערכת לניהול משימות המבוססת על Flask ו-MongoDB.

טכנולוגיות

Backend: Flask, MongoDB

Frontend: JavaScript (AJAX), CSS3

התקנה והרצה

התקנת ספריות:
pip install flask flask-pymongo python-dotenv


יצירת קובץ .env עם הגדרת MONGO_URI.

הרצה: python app.py.

מבנה הקוד

app.py: הרצת השרת.

db.py: חיבור למסד הנתונים.

tasks.py: לוגיקת ה-API (CRUD).

errors.py: טיפול בשגיאות.

static/: קבצי JS ו-CSS.
